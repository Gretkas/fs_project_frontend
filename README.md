# fs_project_frontend
## Installasjon
Tatt fra systemutvikling 2 prosjekt, da installasjon vil fungere på samme måte 

Du trenger:

Git-klient

Node.js og NPM

/* I terminalen */
```
git clone https://github.com/Gretkas/fs_project_frontend.git
cd fs_project_frontend
npm install
npm start
```

Siden vil nå åpnes på localhost:3000.
